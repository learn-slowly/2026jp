# AI Coding Assistant Guidelines (GEMINI.md)

이 문서는 Gemini를 포함한 AI 코딩 어시스턴트가 이 저장소(repository)에서 작업할 때 우선적으로 참고해야 할 전역 규칙과 컨텍스트를 담고 있습니다. AI는 스크립트를 작성하거나 코딩 작업을 수행하기 전 항상 이 문서를 숙지해야 합니다.

---

## 1. 커뮤니케이션 및 깃(Git) 규칙 (CRITICAL)
가장 중요하게 지켜야 할 언어 설정입니다.
* **사용자와의 모든 대화:** 반드시 **한글(Korean)**을 사용하여 명확하고 친절하게 답변합니다.
* **Git 커밋 메시지:** GitHub 커밋 메시지를 작성할 때는 반드시 **한글(Korean)**로 작성합니다. 
  *(작성 예시: `feat: 새로운 후보자 검색 컴포넌트 추가`, `fix: 스타일 렌더링 버그 수정`)*

## 2. 프로젝트 개요
* 이 프로젝트는 **2026 지방선거 디지털 캠페인 플랫폼 (justice-election-2026)** 프로젝트입니다.
* Turborepo를 적용한 Next.js 모노레포 구조 아키텍처를 사용 중입니다 (`apps/2026`, `apps/card`, `apps/page`, `packages/api-client` 등).

## 3. 코딩 컨벤션
* **스타일링:** Tailwind CSS v4를 활용하여 일관된 디자인 시스템을 구축합니다.
* **폰트:** 프로젝트 전반에 기본 폰트로 `Pretendard Variable` (CDN 가변 다이나믹 서브셋)을 적용하여 사용합니다.
* **프레임워크/언어:** 최신 React 문법과 Next.js App Router 생태계를 활용하며 무조건 TypeScript(엄격한 타입 체크)를 적용합니다.

## 4. 작업 가이드라인
* 수정해야 할 파일이 여러 앱(apps)과 패키지(packages)에 걸쳐있을 경우, 해당 구조를 면밀하게 분석한 뒤에 모노레포 규칙에 어긋나지 않도록 의존성을 관리하며 작업합니다.
* UI 작성 시 사용자 경험(UX/UI 개선)을 중심으로 프리미엄하고 모던한 느낌이 들 수 있도록 신경 써서 작성합니다.
